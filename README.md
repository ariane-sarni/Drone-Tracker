# Visual-Inertial Drone Tracker

Real-time UAV detection and tracking in C++. Monocular video in, trajectory out. No Python in the hot path.

## Stack
C++

## Phases
1. **Detection**
2. **Tracking**
3. **Pipeline**
4. **Harden**

## Goal
< 50ms end-to-end
No use of dynamic memory.

## Data
VisDrone or UAV123
